# Install MSR single host using Docker Compose

This section describes how to perform a new single-node Mirantis Secure
Registry (MSR) installation and configuration using Docker Compose. By
following the procedure, you will have a fully functioning single-node MSR
installation with SSL encryption.

!!! warning

    Single-host installations should not be put to use in production
    environments. Mirantis strongly recommends that you use this deployment
    method for development or testing purposes only.
