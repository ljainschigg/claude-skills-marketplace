# Install MSR on a Single-Host using Helm

This section describes how to perform a new single-node Mirantis Secure
Registry (MSR) installation and configuration using Helm. By
following the procedure, you will have a fully functioning single-node
MSR installation with SSL encryption.

- [Prerequisites](helm-prerequisites.md)
- [Install Helm](helm-install.md)
- [Create PVC across Kubernetes workers](helm-storage.md)
- [Install standalone MSR](helm-install-msr.md)

!!! warning

    Single-host installations should not be put to use in production
    environments. Mirantis strongly recommends that you use this deployment
    method for development or testing purposes only.
