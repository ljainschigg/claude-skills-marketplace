# Migration Tool 1.4.0

**Release Date:** 16 December 2025

## Addressed Issues

* [MSRH-500] Fixed an issue wherein the **Total count of pages** was  
  incorrectly calculated whenever non-default page sizes were supplied.
* [MSRH-501] Fixed an issue in the replication rule script that could lead to
  the duplicate processing of the replication rules.
* [MSRH-503] Fixed an issue wherein due to the deprecation of the `--argfile`
  argument, the execution of the `migration_replication_status.sh` script
  failed whenever the `--json` option was applied.
* [MSRH-504] Fixed an erroneous validation of the `HARBOR_DEFAULT_GROUP_ROLE`
  configuration and corrected group role–level mapping issues
  that occur when the source instance uses multiple access levels.
* [MSRH-584] Fixed an issue wherein the project migration failed because the
  source MSR 2.x or 3.x registry did not contain any organizations.
