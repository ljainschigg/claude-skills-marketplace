# Setup for MSR with Entra ID OIDC authentication

Use this tutorial to configure OIDC authentication for MSR with
Microsoft Entra ID. You will register an application in Entra ID,
configure OIDC settings in MSR, set up group mapping, and verify that
users can sign in and receive the correct group memberships.
